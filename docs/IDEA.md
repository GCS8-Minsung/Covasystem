## 1. 사업 핵심 요약

**Cova Container System**은 컨테이너 3개(포집·배양·추출 모듈)를 결합한  
**모듈형 탄소포집-미세조류 통합 플랫폼**

| 구분 | 핵심 내용 |
|------|-----------|
| 사업 모델 | Carbon-Capture-as-a-Service (CaaS) — 고객 CAPEX 無, 월 구독료 방식 |
| 핵심 제품 | Cova 3-모듈 컨테이너 시스템 (포집 20ft + 배양 40ft + 추출 40ft) |
| 주요 고객 | EU 수출 비중 20%↑, 연간 CO₂ 직접 배출 500~5,000톤의 중견 제조업체 |
| 1-site CAPEX | 총 10~17억원 (보조금 활용 후 실질 5~10억원) |
| 연 OPEX | 1.9~2.85억원/site → 실질 중앙값 2.3~3.2억원 권고 |
| 수익구조 | 바이오매스 판매 50% + CaaS 구독료 30% + 탄소크레딧 5% + R&D 지원 15% |
| 단위 감축량 | 연 2~10 tCO₂/컨테이너 세트 |
| BEP (중도) | **3.4년** (1-site, K-ETS 3만원·보조 50% 가정) |
| SROI 20yr | **1.34:1** (1-site) → 2.9~3.0:1 (Stage1 × 50 sites) |
| 기술 결론 | 발효·식품음료·바이오가스 배가스 (CO₂ 농도 >80%)에서 공학적으로 성립 |

**핵심 판단**: 발효·식품·음료 산업의 고농도 CO₂ 배가스 활용 + 피코시아닌·아스타잔틴 고부가가치 추출 + K-ETS/CBAM 규제 모멘텀의 **3중 결합**이 사업성의 핵심 조건.

## 2. 문제 정의

### 2-1. 핵심 인사이트 (Mirror Problem)
> 미세조류 산업의 가장 큰 원가 병목(CO₂ 확보)과 산업 제조업의 가장 큰 규제 부담(CO₂ 배출)이  
> **정확히 대칭을 이루는 거울상 문제(mirror problem)**

두 문제를 동시에 풀 수 있는 모듈형 인프라가 시장에 존재하지 않는다 → **Cova의 기회**

### 2-2. 기존 해법의 한계
| 대안 | 한계 |
|------|------|
| 재생 플라스틱 | 반복 재활용 중 열화·BPA 잔류 → 식품용기 사용 불가. 재활용률 27% 불과 |
| 바이오플라스틱 (PLA·PE) | 식량 vs. 연료 논쟁, 원자재 가격 변동성 |
| 미세조류 기반 PHA | OPEX의 15~25%가 CO₂ 확보 비용 |
| 대규모 CCUS 참여 | 연 30만 톤 이상에서만 경제성 → 중소기업 구조적 배제 |
| 탄소 오프셋 구매 | K-ETS 톤당 약 1만원, 실제 감축 없이 비용만 누적 |

### 2-3. 긴급성 검증 (UXMF)
| 기준 | 근거 |
|------|------|
| Urgent (긴급) | CBAM 2026.1.1 시행, K-ETS 4차 계획기간 유상할당 확대 |
| eXpensive (비쌈) | CBAM 비용 8,470억원(2026~2034), 중소기업 부도확률 2.36배 증가 |
| Mandatory (필수) | EU 수출은 CBAM 회피 불가, K-ETS는 법적 의무 |
| Frequent (빈번) | 매년 반복 배출량 보고, 영속적 문제 |

## 3. 제품 & 서비스

### 3-1. Cova 3-모듈 컨테이너 시스템
| 모듈 | 규격 | 핵심 기능 | 주요 사양 |
|------|------|-----------|-----------|
| 1. 포집 모듈 | 20ft | 공장 배가스에서 CO₂ 직접 포집 | 고체흡착제 기반, 일 0.5~2톤 CO₂ 포집 |
| 2. 배양 모듈 | 40ft | 미세조류 광합성 배양 | 플랫 패널 PBR, LED+태양광 하이브리드 |
| 3. 추출 모듈 | 40ft | 바이오매스 고순도 추출 | HPH 파쇄 + 분무건조 + scCO₂ 추출 |

### 3-2. CaaS 서비스 모델
| CaaS 특징 | 내용 |
|-----------|------|
| CAPEX 부담 제거 | 고객은 초기 투자금 없이 월 구독료만 납부 (**월 500~2,000만원/site**) |
| 수익 공유 | 바이오매스 매출의 일정 비율을 고객에게 Rebate |
| 탄소 크레딧 공유 | K-ETS 할당 외 감축분을 CBAM 상쇄에 활용 |

### 3-3. UVP (고유 가치 제안)
> **"CBAM 인증서 구매 대신, 바이오매스 수익으로 돌아오는 탄소포집 — 중소기업이 처음 가질 수 있는 CCUS"**  
> **"탄소 감축을 위한 Shopify"** — 복잡한 인프라를 플러그앤플레이 모듈로 단순화

## 5. 기술 타당성

### 5-1. 타깃 업종 (배가스 CO₂ 농도 기준)
| 업종 | CO₂ 농도 | 적합성 |
|------|----------|--------|
| 발효·에탄올·바이오가스 | 85~99% | ★★★★★ 최우선 타깃 |
| 식품·음료 (주정·양조) | 60~90% | ★★★★ 우선 타깃 |
| 시멘트 | 14~33% | ★★ 전처리 부담 큼 |
| 철강 (BF-BOF) | 20~44% | ★★ 전처리 부담 큼 |
| 석탄 화력 | 12~15% | ★ 비경제적 |
| 천연가스 CCGT | 3~4% | ✗ 비현실적 |

> ⚠️ **엔지니어링 한계**: Cova의 Tier 1 타깃은 **농도 >80% 발효·식품·음료 산업**에 한정

### 5-2. 미세조류 공학적 근거
| 근거 | 내용 |
|------|------|
| CO₂-바이오매스 고정 비율 | 1 kg 건조 바이오매스 = **1.65~2.0 kg CO₂ 고정** |
| Spirulina 상업 생산성 | 20~30 g/m²/day → 40ft 컨테이너에서 연 **2~10t** 실증 |
| Chlorella 고농도 CO₂ 내성 | 90% vol CO₂까지 내성 보고 |

### 5-3. CAPEX·OPEX 기술 검증
| 항목 | Cova 주장 | 평가 |
|------|-----------|------|
| 포집 20ft | 4~6억원 | 공격적 하단. 1,000t/yr 스케일에서 달성 가능 |
| 배양 40ft | 3~4억원 | 중간대역, 가장 방어 용이 |
| 추출 40ft | 2~4억원 | 식품GMP 포함 시 3~5억원으로 상향 필요 |
| 연 OPEX | 1.9~2.85억원 | **2.3~3.2억원으로 상향 권고** |

## 12. 용어 해설 (UI 텍스트 참조)

| 용어 | 설명 |
|------|------|
| CaaS | Carbon-Capture-as-a-Service. 탄소 포집 장비를 구독 방식으로 제공 |
| CBAM | EU 탄소국경조정제도. 2026년 본격 시행, 수입품에 탄소 비용 부과 |
| K-ETS | 한국 배출권거래제. 4차 계획기간 2026~2030 |
| PBR | 광생물반응기. 미세조류 폐쇄형 배양 장치 |
| 피코시아닌 | 스피룰리나에서 추출하는 청색 색소 단백질. 식품·의약품용 고부가 소재 |
| 아스타잔틴 | Haematococcus에서 추출하는 항산화 카로티노이드. EU 합성 금지로 천연 프리미엄 |
| BEP | 손익분기점. Cova 중도 시나리오 3.4년 |
| SROI | 사회적 투자수익률. 1.34:1 = 1원 투자 시 1.34원의 사회적 가치 창출 |
