# Design System Philosophy

## Visual Identity
Our design system is built on a foundation of high-contrast, technical precision and modern ergonomics. The aesthetic leverages a "Fidelity" color approach, ensuring that our core brand colors remain vibrant and true against a deep, professional dark mode background.

## Color Palette
The interface utilizes a sophisticated dark-mode environment:
- **Primary Accent**: A vibrant Green (#4ADE80) used for critical actions and brand recognition.
- **Secondary Accent**: A bright Blue (#60A5FA) for supporting interactive elements and information hierarchy.
- **Surface & Backgrounds**: A deep, neutral-black base (#1A1A1A) paired with dark grey tertiary surfaces (#333333) to provide depth and clear content layering.

## Typography
We employ **Inter** as our universal typeface across headlines, body text, and labels. This choice ensures maximum readability and a clean, modernist look that supports our technical identity.

## Shape & Geometry
The interface prioritizes approachable, soft geometry. With a **maximum roundedness** setting (level 3), components feature pill-shaped buttons and heavily radiused containers, creating a friendly and contemporary feel that contrasts effectively with the technical dark-mode palette.

## Layout & Rhythm
The system uses a **normal spacing** scale (level 2). This provides a balanced density that is neither too cramped nor overly sparse, facilitating efficient information processing while maintaining professional visual breathing room.
