# Test_case_GAME2.md — 공장 시뮬레이션 게임 테스트 시나리오

## 테스트 환경

- **URL**: `http://localhost:5173` (개발 서버)
- **브라우저**: Chrome 최신 버전 권장
- **섹션 위치**: 랜딩페이지 하단 스크롤 → PHASE 05 — SIMULATION
- **전제**: `npm run dev` 실행 후 접속

## TC-001 | 게임 초기 상태 확인

**목적**: 게임이 올바르게 초기화되었는지 확인

**절차**:
1. 랜딩페이지 접속
2. 아래로 스크롤하여 PHASE 05 섹션 진입
3. 게임 캔버스 확인

**기대 결과**:
- [ ] 24×16 그리드 맵이 표시된다
- [ ] 공장(🏭) 아이콘이 2~4개 랜덤 배치되어 있다
- [ ] 공장 간 간격이 충분히 이격되어 있다 (겹침 없음)
- [ ] 하단 툴바에 7종 장치 버튼이 표시된다
- [ ] 진행률 표시가 `0/100`으로 시작한다
- [ ] "탄소 포집기" 버튼이 기본 선택(초록 테두리)되어 있다

## TC-002 | 탄소 포집기 배치

**목적**: 장치 배치 기능 확인

**절차**:
1. 하단 툴바에서 "탄소 포집기(🧲)" 선택
2. 공장 인접 빈 셀 클릭
3. 배치된 셀 확인

**기대 결과**:
- [ ] 클릭한 셀에 탄소 포집기 아이콘이 표시된다
- [ ] 초록빛 배경 하이라이트가 적용된다
- [ ] 공장 셀에는 배치가 불가하다 (클릭해도 변화 없음)

---

## TC-003 | 배관 배치 및 방향 전환

**목적**: 배관 배치 및 방향 전환 확인

**절차**:
1. 툴바에서 "배관(⟶)" 선택
2. 포집기 오른쪽 셀에 배관 배치
3. 같은 셀을 다시 클릭 (방향 전환)
4. 총 4번 클릭하여 방향 순환 확인

**기대 결과**:
- [ ] 배관 배치 시 화살표 기호가 표시된다 (→ ↓ ← ↑ 순환)
- [ ] 첫 배치: `→` 방향
- [ ] 클릭마다 방향이 순환한다 (→ → ↓ → ← → ↑ → →)
- [ ] 우클릭 시 배관이 제거된다

---

## TC-004 | 전체 공정 연결 및 자원 흐름

**목적**: 아이템이 파이프라인을 따라 올바르게 흐르는지 확인

**절차**:
1. 공장 옆에 탄소 포집기 배치
2. 포집기 → 정제기 방향으로 배관 연결
3. 배관 끝에 정제기(⚗️) 배치
4. 정제기 출력부에 두 개의 배관 연결
   - 한 배관은 배양기(🌱)로
   - 다른 배관은 쓰레기통(🗑️)으로
5. 배양기 출력을 제조기(⚙️)로 연결
6. 제조기 출력을 저장 창고(📦)로 연결
7. 시간 경과 관찰

**기대 결과**:
- [ ] 공장에서 회색 점(매연)이 나타난다
- [ ] 매연이 배관을 따라 이동한다
- [ ] 포집기에서 짙은 회색 점(포집 매연)으로 변환된다
- [ ] 정제기에서 초록 점(CO₂)과 주황 점(중금속)으로 분리된다
- [ ] CO₂는 배양기 방향으로, 중금속은 쓰레기통 방향으로 이동한다
- [ ] 쓰레기통에 중금속이 도달하면 소멸한다
- [ ] 배양기에서 청록 점(수분 조류)이 생성된다
- [ ] 제조기에서 진초록 점(바이오매스)이 생성된다
- [ ] 저장 창고에 바이오매스가 도달하면 진행률이 증가한다

---

## TC-005 | 배양기 사멸 인터랙션

**목적**: 정제되지 않은 매연이 배양기에 공급될 때 사멸 인터랙션 확인

**절차**:
1. 공장에서 배양기로 직접 배관 연결 (정제기 거치지 않음)
2. 연결 후 시간 경과 관찰

**기대 결과**:
- [ ] 매연(회색)이 배양기에 도달하면 배양기가 빨간색으로 변한다
- [ ] 💀 이모지가 잠시 표시된다
- [ ] 아이템이 소멸하며 출력이 없다
- [ ] 몇 초 후 배양기가 정상 상태로 돌아온다
- [ ] CO₂를 공급하면 다시 정상 작동한다

## TC-007 | 승리 조건 확인

**목적**: 저장 창고에 100개 누적 시 승리 화면 표시 확인

2. 저장 창고에 바이오매스가 쌓이도록 대기
3. (빠른 테스트) GameEngine.js의 STORAGE_GOAL을 5로 임시 변경 후 테스트

**기대 결과**:
- [ ] 진행률 바가 증가한다 (예: `23/100`)
- [ ] 저장 창고에 아이템 도달 시 초록 글로우 이펙트가 표시된다
- [ ] 100개 달성 시 승리 오버레이가 표시된다
- [ ] 오버레이에 "미션 완료!" 텍스트와 🎉 이모지가 표시된다
- [ ] "다시 하기" 버튼 클릭 시 새 게임이 시작된다

## 빠른 테스트 팁

전체 공정을 빠르게 테스트하려면 다음 레이아웃을 권장합니다:

```
공장(고정) → 포집기 → 배관×3 → 정제기 → 배관(CO₂방향) → 배양기 → 배관×2 → 제조기 → 배관 → 창고
                                         └→ 배관(중금속방향) → 쓰레기통
```

승리 조건을 빠르게 확인하려면:
- `src/components/game2/gameItems.js` 파일의 `STORAGE_GOAL`을 `5`로 임시 변경
- 테스트 후 `100`으로 복원
