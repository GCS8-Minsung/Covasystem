# PRD — Atmospheric Vitality 랜딩페이지

## 1. 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | Atmospheric Vitality |
| 목적 | GCS(Green Carbon Solution) 사업을 소개하는 인터랙티브 랜딩페이지 |
| 핵심 가치 | 탄소 포집 → 미세조류 산소 생성 순환 프로세스를 시각적·체험적으로 전달 |
| 주요 대상 | 잠재 투자자, 파트너사, 일반 사용자 |

---

## 2. 목표

1. **브랜드 인지** — 다크 글래스모피즘 디자인으로 고급스러운 ESG 기술 기업 이미지 구축
2. **콘텐츠 전달** — 스크롤 기반 스토리텔링으로 탄소 포집 → 미세조류 → 산소 순환 설명
3. **인터랙션 체험** — 웹캠 손 인식 또는 마우스로 직접 탄소를 포집하는 미니게임 제공
4. **다국어 지원** — 한/영 실시간 전환

---

## 3. 기능 요구사항

### 3.1 랜딩페이지 (스크롤 기반 단일 페이지)

#### FR-01: 최초 접속 — Hero 섹션
- 전체화면 Hero 레이아웃으로 첫 진입
- 비대칭 타이포그래피: "탄소, 새로운 생명의 시작으로" (한) + "Carbon, to the Start of New Life." (영)
- 배경: 다크 인더스트리얼 텍스처 + 주변광 그라디언트
- 하단 스크롤 큐 애니메이션(bounce) 표시

#### FR-02: 스크롤 애니메이션
- 각 섹션 진입 시 콘텐츠가 동적으로 등장 (fade-up + stagger)
- Intersection Observer 기반, 최초 진입 시 한 번만 실행

#### FR-03: Phase 01 — Capture the Transition
- 탄소 포집 기술 소개 벤토 그리드
- 대형 카드: 직접 공기 포집(DAC) 코어 + Air→Science 플로우
- 메트릭 카드: 효율 94%, 스마트 그리드 연동

#### FR-04: Phase 02 — Algae & Oxygen
- 미세조류 소개 벤토 그리드
- 광생물반응기 이미지 카드 (hover 확대 효과)
- 메트릭: O₂ 생산율 98.4%, Biomass 부산물 설명
- 원형 프로그레스 게이지: 스크롤 진입 시 0→82% 애니메이션

#### FR-05: Phase 03 — Interactive Algae HUD
- 시뮬레이션 뷰어(16:9) + 사이드 데이터 패널
- 실시간 지표: 산소 94% 순도, 탄소 흡수 1,204 KG
- HUD 코너 액센트, 라이브 피드 표시
- 미니게임 섹션으로의 CTA 버튼

#### FR-06: 미니게임 — Carbon Capture Game
- 화면 전체 Canvas 기반 게임 (스크롤 최하단)
- 탄소 버블(CO₂ 분자)이 화면 안을 랜덤하게 유영
- **웹캠 모드**: MediaPipe Hands로 손 랜드마크 인식
  - 총 모양(검지 펼침) → 조준 상태 + 조준선 표시
  - 주먹 쥐기 → 포집 빔 발사 (검지 방향 벡터 기준)
- **마우스 폴백**: 웹캠 거부/없을 시 마우스 이동=조준, 클릭=발사
- 스코어 HUD: 포집한 탄소량(KG) 누적 표시

### 3.2 공통 기능

#### FR-07: Navbar
- 고정 상단 바, 스크롤 시 배경 blur 활성화
- 로고, 섹션 링크(앵커 스크롤), 한/영 토글, CTA 버튼
- 모바일 햄버거 메뉴

#### FR-08: 언어 전환
- 상단 KO/EN 토글 버튼
- 전체 UI 텍스트 실시간 전환 (React Context)

#### FR-09: 스크롤 진행 표시
- 화면 최상단에 얇은 프로그레스 바

## 4. 비기능 요구사항

| 항목 | 요구사항 |
|------|---------|
| 성능 | 게임 루프 60fps 유지, Lighthouse 성능 점수 80 이상 |
| 접근성 | 웹캠 없이도 마우스 폴백으로 전체 기능 사용 가능 |
| 보안 | 웹캠 권한은 사용자 명시적 동의 후에만 요청 (MediaPipe는 HTTPS 필요) |
| 반응형 | 모바일(375px) / 태블릿(768px) / 데스크탑(1280px+) 대응 |
| 브라우저 | Chrome 110+, Firefox 115+, Safari 16+ |
| 배포 | Vercel 또는 Netlify (HTTPS 자동 제공) |
